Website Snippet Designer and Builder Odoo App

Build Custom Snippet Odoo Apps helps users to easily build custom snippets for websites without any technical knowledge. It is very useful to reuse added snippets again, users can see added snippet details on the backend menu.

Features

Add Custom Website Snippet.

User can easily create custom website snippet from website.

Create Once Use Anywhere Anytime.

User have to just create snippet once and can reuse multiple time at anywhere on website.

Manage Snippet from Back-End.

User can also create snippet from back-end or change code for snippet.

Drag and Drop, Easily to Use.

User have to drag and drop snippet, can easily use, no need configure.
Demo Request Video Tutorial

Website Snippet Builder

User have to click "Edit" button on website.

Here user can see "Snippet Builder" under snippet builder block, User need to drag and drop snippet builder on website where he/she wants to build custom snippet.

Snippet Builder

On "Snippet Builder" wizard user have to give name for snippet, and also add HTML, CSS and JS code for snippet and click on "Make Snippet" button.

Custom Website Snippet

User can see added custom snippet to website using snippet builder.

Onces user created custom snippet it will added to blocks and can reuse snippet anywhere on website by just drag and drop.

Website Snippet Menu

Under Website > Configuration > Website Snippet menu user can create website snippet, User can also see created website snippet in tree view.

Website Snippet Builder Form View

User can see added HTML, CSS and JS code for snippet and also can change code for snippet.

Google Tag Manager Odoo App helps to configure container id in odoo website. Users can easily manage all tags without editing code with using google tag manager. Users can also add tags for measuring product impressions, product clicks, views of product details, additions to a shopping cart, checkout and purchases. Users can easily create and configure tags in google tag manager for tracking website data.
